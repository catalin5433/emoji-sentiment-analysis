This project explores the relationship between emojis and sentiment using a dataset of 1.6 million tweets collected across 13 European languages(data source: https://kt.ijs.si/data/Emoji_sentiment_ranking/index.html). The dataset contains precomputed sentiment distributions (positive, neutral, negative) for each emoji, derived from a large corpus of labelled tweets. 
You can test the notebook interactively in your browser (no installation required) via MyBinder: https://bit.ly/4teoer4 .

The project enables answering questions such as:

- Which emojis are most strongly associated with positive sentiment?
- Which emojis tend to convey negative emotions?
- How frequently are emojis used in tweets?


This project highlights:

- Data cleaning and preprocessing
- Feature engineering
- Exploratory data analysis (EDA)
- Working with real-world datasets


It was developed using Python, Pandas, and Jupyter Notebook.
